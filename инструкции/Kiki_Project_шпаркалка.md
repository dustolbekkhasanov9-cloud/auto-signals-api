# Kiki Auto Signals --- Полная инструкция

## Часть 1 --- Как устроен проект

Проект состоит из двух частей:

### 1. iPhone приложение (SwiftUI)

Файлы находятся в Xcode проекте.

Главные файлы:

ContentView.swift --- интерфейс приложения\
KikiApp.swift --- точка входа приложения

Приложение: - показывает торговые сигналы - строит графики - показывает
цену входа и выхода - показывает время входа и выхода - получает данные
из API в облаке

------------------------------------------------------------------------

### 2. Облачный сервер (Python)

Работает на Render.

Основные файлы backend:

api.py --- API сервер\
signal_engine.py --- логика сигналов\
requirements.txt --- библиотеки Python\
render.yaml --- настройки деплоя

Backend получает рыночные данные, рассчитывает сигнал и отправляет JSON
приложению.

------------------------------------------------------------------------

# Архитектура системы

iPhone App\
↓\
Render API\
↓\
signal_engine.py\
↓\
рынок / расчёты\
↓\
JSON → обратно в приложение

# Где что редактировать

## Изменить дизайн приложения

Редактировать:

ContentView.swift

После изменений:

Command + B

Если ошибок нет --- запускать приложение.

------------------------------------------------------------------------

## Изменить логику сигналов

Редактировать:

signal_engine.py

или

api.py

После этого нужно обновить сервер в облаке.

------------------------------------------------------------------------

# Как обновить сервер (Render)

Открой терминал и перейди в папку backend:

cd Облако

Затем выполнить:

git add . git commit -m "update" git push

После push Render автоматически сделает deploy.

------------------------------------------------------------------------

# Проверка работы сервера

Открыть в браузере:

https://auto-signals-api.onrender.com/

Проверить сигналы:

https://auto-signals-api.onrender.com/signals

Проверить одну пару:

https://auto-signals-api.onrender.com/signal?symbol=EURUSD=X

------------------------------------------------------------------------

# Если приложение не показывает данные

Проверить:

1.  apiBaseURL в ContentView.swift

Должно быть:

https://auto-signals-api.onrender.com

2.  Проверить API через браузер

3.  Проверить ошибки Xcode

------------------------------------------------------------------------

# Если Xcode показывает Build Failed

Чаще всего проблема в ContentView.swift.

Проверь:

-   лишнюю }
-   недостающую }
-   вложенные struct
-   дублирование struct

------------------------------------------------------------------------

# Если ошибка

Cannot find 'ContentView' in scope

Значит:

ContentView.swift не компилируется.

Нужно исправить ошибки в нём.

------------------------------------------------------------------------

# Если время входа/выхода неправильное

Backend отдаёт:

entry_time_iso exit_time_iso

Приложение переводит время в локальный часовой пояс телефона.

Проверить функцию:

localTimeString()
